# EVALUACIÓN FINANCIERA V1 APROBADA

Diseño aprobado para el módulo Evaluación Financiera de NEXUS PRO.

## Flujo
Cliente → Evaluación Financiera → Aprobación → Préstamo → Cobranza.

## Componentes
- Información del cliente
- Información económica
- Análisis financiero
- Simulador de préstamo en tiempo real
- Score interno
- Recomendación automática
- Resumen lateral fijo
- Botón principal: Aprobar y crear préstamo
